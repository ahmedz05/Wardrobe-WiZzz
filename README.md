# Wardrobe-WiZzz
AI-powered wardrobe management and outfit recommendation platform using computer vision, fashion intelligence, and personalized styling.

## Features

- Smart wardrobe organization
- Clothing image analysis
- Automatic categorization
- Outfit generation
- Fashion rule engine
- Color theory recommendations
- Personalized styling
- Brand and collection metadata
- AI stylist assistant

## Tech Stack

### Frontend
- Flutter

### Backend
- Python
- FastAPI

### Database
- PostgreSQL

### AI
- OpenCV
- YOLO
- EasyOCR
- LLM APIs
